### Installation

First, clone the repo

```sh
git clone https://github.com/jimmypangalinan/hello-world-express-js.git
```

move to the folder

```sh
cd hello-wolrd-express-js
```

install dependencies:

```sh
npm install
```

execute the project

```sh
npm start
```
